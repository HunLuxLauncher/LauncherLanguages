# Translations for [HunLux Launcher](https://hunluxlauncher.hu/en-us/launcher)

Feel free to fork it and make a translation if you're language is not already translated or search issues to find out if your language has pending translations.

- [Hungarian (magyar)](/hu-HU.xml)
